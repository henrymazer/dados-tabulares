# IBGE: Ingestao Espacial Base

## Parent PRD

**PRD-05.7-ingestao-espacial-ibge.md**

## What to build

Implementar uma fatia tecnica de fundacao para ingestao espacial do IBGE, provando parser, persistencia espacial e fluxo ETL minimo para popular `ibge.setores_censitarios` com `MultiPolygon` em SRID `4674`. Este item nao representa mais a ingestao real dos arquivos oficiais do IBGE; essa trilha foi desdobrada posteriormente nos work-items `05.10`, `05.11`, `05.12`, `05.13` e `05.14`.

## Acceptance criteria

- [ ] Existe um dataset ETL explicito para setores censitarios do IBGE
- [ ] O parser converte `Polygon` e `MultiPolygon` para `MultiPolygon`
- [ ] A carga substitui integralmente `ibge.setores_censitarios` sem afetar datasets tabulares
- [ ] Geometria invalida ou tipo espacial nao suportado falha com mensagem clara
- [ ] Existem testes para parser, command parser e ingestao integrada
- [ ] O work-item deixa explicito que serve como fundacao tecnica para a trilha de ingestao real criada a partir do `05.8`
