# Geocodificação Nominatim com Decisão Automática

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Executar a geocodificação primária com Nominatim self-hosted usando as queries definidas no PRD, armazenando tentativas, candidatos e decisão automática inicial baseada em consistência espacial com o município esperado.

## Acceptance criteria

- [ ] O pipeline consulta o Nominatim com as duas estratégias de query definidas
- [ ] A decisão automática aceita apenas candidatos válidos dentro do município esperado
- [ ] O histórico de tentativas e candidatos fica persistido
- [ ] Casos ambíguos, fora do município ou não resolvidos recebem status operacional adequado
