# Snapshot e Auditoria de Cargas Brutas

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Criar a base transversal de rastreabilidade das ingestões, registrando origem, arquivo, hash, snapshot e metadados suficientes para idempotência e reprocessamento seguro. O slice deve ser reutilizável pelos pipelines do IBGE e do TSE.

## Acceptance criteria

- [ ] Existe persistência de metadados de snapshot por carga
- [ ] Reprocessar a mesma fonte com o mesmo snapshot não duplica dados
- [ ] Os pipelines conseguem identificar qual arquivo bruto originou a carga
- [ ] Testes cobrem comportamento de idempotência e substituição de snapshot
