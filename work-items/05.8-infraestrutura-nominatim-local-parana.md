# Infraestrutura Nominatim Local Paraná

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Preparar uma fatia operacional completa para rodar o Nominatim localmente com foco inicial no extrato do Paraná. O slice deve cobrir infraestrutura, configuração segura e fluxo mínimo de uso para que os próximos slices de geocodificação possam depender de um serviço funcional e reproduzível.

## Acceptance criteria

- [ ] Existe infraestrutura local versionada para subir o Nominatim em modo de desenvolvimento
- [ ] O fluxo documentado cobre import inicial do extrato do Paraná
- [ ] Segredos e configuração operacional ficam fora de arquivos versionados
- [ ] Há uma forma verificável de consultar o serviço local após a importação
