# Consolidação Canônica de Locais de Votação TSE

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Consolidar os registros do TSE no nível de local de votação usando a chave canônica definida, gerando uma base operacional única para geocodificação e posterior vinculação dos fatos eleitorais.

## Acceptance criteria

- [ ] Os registros do TSE são consolidados no nível de local de votação
- [ ] A identidade canônica usa `SG_UF + CD_MUNICIPIO + NR_ZONA + NR_LOCAL_VOTACAO`
- [ ] O resultado consolidado referencia corretamente o staging bruto
- [ ] A consolidação pode ser reexecutada sem gerar duplicações
