# IBGE: Modelagem Espacial

## Parent PRD

**PRD-05.6-modelagem-espacial-ibge.md**

## What to build

Implementar a fundação espacial do IBGE no schema `ibge`, com a primeira entidade geográfica canônica para setores censitários, o mapeamento EF/PostGIS correspondente, o SRID definido e os índices espaciais necessários. Este slice estabelece a forma do dado, mas não ingere arquivos nem expõe endpoints.

## Acceptance criteria

- [x] Existe uma entidade espacial do IBGE mapeada para o schema `ibge`
- [x] A geometria usa um SRID canônico definido para o projeto
- [x] Índices espaciais são criados para as colunas geográficas
- [x] As migrations foram atualizadas para aplicar no banco local com PostGIS
- [x] Os vínculos com município/UF permanecem consistentes com o modelo atual
- [x] Não existe ingestão espacial nem endpoint novo neste slice
