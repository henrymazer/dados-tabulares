# Definição Assistida do Schema Canônico do TSE e Catálogo Derivado

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Conduzir uma etapa assistida com o usuário para definir quais colunas do staging do TSE sobem para a camada canônica e gerar o catálogo derivado do schema observado, mantendo foco no nível de local de votação e descartando detalhes sem valor analítico principal.

## Acceptance criteria

- [ ] O procedimento assistido pergunta ao usuário quais campos do TSE permanecem no schema canônico
- [ ] O schema canônico do TSE fica definido no nível de local de votação
- [ ] O catálogo derivado do TSE é persistível e rastreável ao schema observado
- [ ] A decisão explicita o papel de campos mantidos só em staging
