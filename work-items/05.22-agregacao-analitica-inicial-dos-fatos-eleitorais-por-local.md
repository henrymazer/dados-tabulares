# Agregação Analítica Inicial dos Fatos Eleitorais por Local

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Gerar a primeira fatia analítica útil dos fatos eleitorais agregados por local de votação, usando a geocodificação final e os fatos canônicos já vinculados para permitir consultas territoriais no grão que faz sentido ao usuário.

## Acceptance criteria

- [ ] Existe uma agregação inicial verificável dos fatos eleitorais por local
- [ ] A agregação usa apenas locais com status espacial final definido
- [ ] O resultado fica pronto para consumo posterior por API e análises territoriais
- [ ] Testes cobrem consistência entre fatos canônicos, local e agregação resultante
