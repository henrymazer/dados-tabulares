# IBGE: Ingestao Espacial

## Parent PRD

**PRD-05.7-ingestao-espacial-ibge.md**

## What to build

Implementar a ingestao espacial de setores censitarios do IBGE, adicionando um parser GeoJSON dedicado e o fluxo ETL necessario para popular `ibge.setores_censitarios` com `MultiPolygon` em SRID `4674`.

## Acceptance criteria

- [ ] Existe um dataset ETL explicito para setores censitarios do IBGE
- [ ] O parser converte `Polygon` e `MultiPolygon` para `MultiPolygon`
- [ ] A carga substitui integralmente `ibge.setores_censitarios` sem afetar datasets tabulares
- [ ] Geometria invalida ou tipo espacial nao suportado falha com mensagem clara
- [ ] Existem testes para parser, command parser e ingestao integrada
