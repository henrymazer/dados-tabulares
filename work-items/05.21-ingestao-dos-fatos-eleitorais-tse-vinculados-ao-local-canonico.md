# Ingestão dos Fatos Eleitorais TSE Vinculados ao Local Canônico

## Parent PRD

**PRD-05.8-ingestao-real-geoespacial-ibge-tse.md**

## What to build

Ingerir os fatos eleitorais do TSE no menor grão útil do arquivo, vinculando cada registro ao local canônico já consolidado para preservar flexibilidade analítica e permitir agregações posteriores por local.

## Acceptance criteria

- [ ] Os fatos eleitorais do TSE são ingeridos a partir dos arquivos reais
- [ ] Cada fato fica vinculado corretamente ao local canônico correspondente
- [ ] O grão canônico mínimo do arquivo é preservado antes de agregações
- [ ] Testes cobrem vínculo correto e idempotência da carga
