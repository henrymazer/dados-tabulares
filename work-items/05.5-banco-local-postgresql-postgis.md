# Banco Local PostgreSQL + PostGIS

## Parent PRD

**PRD-05.5-banco-local-postgresql-postgis.md**

## What to build

Criar a infraestrutura de desenvolvimento local para subir um PostgreSQL com PostGIS e deixar a configuração do `Data` pronta para usar o provider espacial do Npgsql. Este slice cobre apenas a base local e a preparação técnica para futuras colunas geográficas.

## Acceptance criteria

- [x] Existe um arquivo versionado para subir o banco local com PostGIS
- [x] O banco local cria a extensão `postgis` automaticamente
- [x] O ambiente local usa volume persistente
- [x] O projeto `Data` fica preparado para NetTopologySuite no provider Npgsql
- [x] Há instrução curta no repositório explicando como subir o banco local e qual conexão usar no desenvolvimento
